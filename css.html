CSS (Cascading Stylesheet Document) is a more organized way of seperating your style of your website from the html doccument. Then you can link it to your html doccument so that all of the content can come together. Its a style sheet language used for describing the look and formatting of a document written in a markup language. Its most common application is to style web pages written in HTML.
